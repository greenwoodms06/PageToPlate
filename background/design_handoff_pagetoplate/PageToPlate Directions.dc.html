5AI import help — the copy-paste workflow from 4a
5aImport a book with AI help — numbered steps, copyable prompt
Import a book with AI help
Skip typing hundreds of recipes — let an AI chatbot read your book’s table of contents and build the file for you. Takes about 5 minutes.
1
Photograph the table of contents
Or the index — whichever lists recipe names with page numbers. Several photos are fine.
example: photo of a contents page
2
Paste this prompt into any AI chatbot
Attach your photos, then send the prompt (Claude, ChatGPT, Gemini — any works).
Read the attached cookbook table-of-contents photos. Output a CSV with columns: recipe name, page, category. Use these categories where possible: Mains, Sides, Soups & Stews, Salads… (+8 more lines)
3
Save the reply as a .csv file
Copy the chatbot’s output into a text file named anything ending in .csv. Don’t worry about stray spaces — the importer is forgiving.
4
Import it here
You’ll get a preview (“47 recipes found”) before anything is saved. Re-importing later never duplicates recipes.
Names + page numbers only — no recipe text is copied. Fix any wrong category later, right on the recipe card.
